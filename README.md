# Eros - Next.js App

This is a Next.js application migrated from Vite.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Routes

- `/` - Dashboard (main app)
- `/landing` - Landing page

## Build for Production

```bash
npm run build
npm start
```
